# Formatters Library

Utility # @cazes/formatters

A lightweight, framework-agnostic utility library for formatting and validating common data types such as phone numbers, postal codes, NAS, and emails.

This library is designed to be:
- ⚡ Fast
- 🧠 Simple
- 🔌 Framework-agnostic (React, Node, Vanilla JS)
- 📦 Tree-shakable

---

## 📦 Installation

```bash
npm install @cazes/formatters
